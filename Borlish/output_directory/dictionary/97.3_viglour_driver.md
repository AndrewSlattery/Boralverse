97.3: viglour "driver"

viglour /vajˈlur/ [vɐjˈlur]
- driver, person operating a vehicle;
- chauffeur, person employed to operate a vehicle;
- instigator, cause, someone or something which provokes or sets in motion

Etymology: agentive formation from viglar "to drive (a vehicle); to set in motion", from now-obsolete vigl "cart, wagon". This descends via Old Borlish veiȝl from Latin vehiculum "means of transport". In metaphorical use attested from the sixteenth century.
