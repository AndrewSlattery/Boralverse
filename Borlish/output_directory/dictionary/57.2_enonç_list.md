57.2: enonç "list"

enonç /eˈnɔnts/ [ɪˈnons]
- list, enumeration, the compilation of a set of items;
- order, ordering, a particular arrangement of a given set in sequence

Etymology: deverbal noun from enonçar "enumerate, list, order", orignally "make a report". This descends via Old Boral from Latin ēnūntiō "I disclose, express, report".
