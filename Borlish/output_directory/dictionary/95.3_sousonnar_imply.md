95.3: sousonnar "imply"

sousonnar /ˌsu.zɔˈnar/ [ˌsu.zʊˈnɑː]
- imply, suggest, intimate, hint, to express indirectly through implication;
- to write in code or metaphor

Etymology: Middle Borlish borrowing from Latin subsonō "I express secretly".
