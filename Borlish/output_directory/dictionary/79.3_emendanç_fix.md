79.3: emendanç "fix"

emendanç /ˌe.mɛnˈdants/ [ˌe.mɪnˈdans]
- fix, repair, something done to mend a broken object or situation;
- resolution, solution, answer, something that solves a problem

Etymology: noun derived from emendar "to fix, repair, resolve, sort out", which descends (via Old Boral amendar, later re-classicised) from Latin ēmendō "I improve, remedy, cure".
